* Usage:
  >*cdls* ***dir_name***

  | *cdls* | Move to /home/$USER |

  |*cdls* ***.*** |  Update current directory |

  |*cdls* ***..*** |  Move up one directory |

  |*cdls* ***-*** |  Move to previous directory |

Set *CLEAN_TTY* to **True** if you want to enable clearing terminal window while using **cdls**
Set **False** to disable
